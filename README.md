Niek Brot
Niek Brot is een interactieve chatbot die leuke weetjes deelt, nieuwe feiten opslaat en rustgevende muziek afspeelt binnen een gesprek.
Functie's
-Random feitjes (opslag): Geef willekeurig feit uit de database
-Feitjes toevoegen (Wist ik al): nieuwe feitjes toevoegen
-Rustige Niek muziek (media): Start de bekende rustige achtergrondmuziek tijdens het gesprek.

Fucties aanroepen
-!niek
-!niekadd [feitje]
-!nieknee
Feiten
- Plastice deetjes in de flessen
- wist je dat een vliegtuig meer uitstoot heeft dan een trein
Meer vliegtuigen in de zee dan onderzeeris in de lucht
